# 🚢 Real-Time Ferry Ticket Sales & Redemption Analytics for Toronto Island Park

## 📌 Project Overview

This project focuses on analyzing real-time ferry ticket sales and redemption data for Toronto Island Park. The objective is to identify passenger demand patterns, peak travel periods, seasonal trends, and operational opportunities using data analytics and visualization techniques.

The project combines Exploratory Data Analysis (EDA), statistical testing, KPI tracking, and interactive dashboard development to support data-driven decision-making for ferry scheduling, passenger flow management, and operational planning.

---

## 🎯 Project Objectives

### Primary Objectives
- Analyze historical and near real-time ticket activity
- Identify peak and off-peak ferry demand periods
- Quantify passenger inflow and outflow trends
- Track ticket sales and redemption patterns
- Generate operational insights for ferry scheduling

### Secondary Objectives
- Support data-driven decision-making
- Improve passenger experience during peak periods
- Optimize resource allocation and staffing
- Enhance transportation planning efficiency

---

## 📊 Dataset Description

The dataset contains ferry ticket sales and redemption records collected at 15-minute intervals.

| Column | Description |
|----------|------------|
| _id | Unique row identifier |
| Timestamp | Date and time of ticket activity |
| Sales Count | Number of tickets sold |
| Redemption Count | Number of tickets redeemed |

## 🔍 Key Findings

### Demand Trends
- Passenger demand varies significantly across hours, days, and seasons.
- Summer season records the highest passenger activity.
- August is identified as the busiest month.

### Passenger Flow
- Ticket sales and redemptions show a strong positive correlation.
- Passenger movement trends help identify periods of congestion and resource requirements.

### Operational Insights
- Peak demand periods require increased ferry frequency.
- Seasonal trends indicate tourism-driven demand fluctuations.
- Weekend travel behavior differs from weekday travel patterns.

### Statistical Findings
- Pearson correlation analysis confirms a strong relationship between ticket sales and ticket redemptions.
- Statistical significance testing validates that the observed relationship is meaningful and not due to random variation.
